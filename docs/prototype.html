  • 推力：西工大 −1.2% / 六院 −2.7% / 北航 −11%（越界）
  • 燃烧效率：西工大 −0.7% / 六院 −1.7% / 北航 −5.7%（偏低）
  • 其余（室压/比冲/质量流量）西工大与六院均 ≤3%，北航 5–10%

三、收敛与网格质量

三算例均满足稳态收敛判据。网格：西工大 12M/y+35（优）、六院 8M/y+55（良）、北航 6.4M/y+88（偏粗）。

四、专家结论（渊）

西工大 case4 各项偏差 ≤2.5%，判定可信；六院次之；北航存在 ~10% 系统性偏高，推力越界 ［判据·文献］，疑似入口总压设定偏差。

五、评级与建议

综合评级：西工大 A− > 航天六院 B+ > 北航 B。建议北航复核入口总压、加密近壁网格后重交付。

快速上手

平台的主动线是 入库 → 对比 → 报告：

1. 数据资源库 批量入库各单位交付（仿真 HDF5 / 实验 TXT），系统自动解析、去重、对齐工况；
2. 勾选要评的对象 → 对比评估，按工况/算例/单位三个粒度看偏差与排名；
3. 一键 生成评估报告，导出或追加到知识库。

需要深入一个算例时，去 仿真分析 / 试验分析（加工层，产出对比评估用的 QOI 与真值）。

数据规范（入库前必读）

实验数据 TXT：逗号分隔、UTF-8；表头在第 headerIndex 行（默认 10，可在配置改）；第 0 列为 Time (s)；通道命名遵循 流道\d+ / 室压\d+ / 隔离段\d+（规则可配置）；压力如为相对值，入库时声明大气压修正。

仿真数据 HDF5：Fluent *.cas.h5 + *.dat.h5 成对提交；当前支持单网格 meshes/1 单相 phase-1；入口 zone 名含 inlet。

⚠️ 工况标识规范（多源对比的前提）：每份交付必须提供工况标识——来流 Ma / 动压 / 总温 或统一工况编号。缺失或无法映射的，入库后标 PENDING，需人工在资源库指定工况才能进对比。这是"同一工况才能比"的地基。

功能说明

各模块（资源库 / 对比评估 / 仿真分析 / 试验分析 / 评估报告 / RAGflow / 配置）的详细用法。（阶段一补全）

评估方法学（评分如何得出 · 透明可追溯）

偏差：各单位物理量相对实验真值的百分比偏差。

判据：读 sim-knowledge/criteria.yaml（护城河判据库，带文献引用），非写死阈值。如推力越界判据 is_thrust_out_of_range。

置信度：HIGH（判据/真值直接命中）· MED（规则推断）· LOW（弱证据）· PENDING（待人工）。

评级：由工况级（L3）偏差 + 算例级（L2）质量加权聚合到单位级（L1），权重方案可配置。
